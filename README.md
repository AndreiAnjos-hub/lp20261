# Execução de Código

python3 nome_do_arquivo.py : Roda o seu script Python no terminal. É o comando de "dar o play" no código.

# Fluxo do Git

git add nome_arquivo : Prepara o arquivo para o salvamento. É como colocar o item dentro da caixa antes de fechar

git status : Mostra o estado atual: quais arquivos foram modificados ou ainda não foram salvos.

git commit -m "Descreva aqui o que foi realizado" : Salva oficialmente as alterações no seu computador com uma etiqueta descrevendo o que foi feito.

git push : Envia todos os seus salvamentos (commits) do computador para o servidor online (como o GitHub).

git rm --cached nome_do_arquivo_antigo.extensao : Exclui um arquivo a força do repositório.
